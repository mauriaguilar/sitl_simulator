# sitl_simulator

## Clone Simulator repo
* git clone https://gitlab.unc.edu.ar/lc-investigacion/drone/simulator
* cd simulator
* Build: make build
* Launch: make

## Autopilot
* Install Autopilot on autopilot2 folder.
* Copy the Makefile inside that folder.
* cd autopilot2
* make
* The autopilot will be launched.
